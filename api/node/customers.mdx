## Overview

The `CustomersClient` provides full CRUD operations for managing end-user (customer) records in your application.

## Methods

### identify()

Identify (upsert) a customer. Creates the customer if not found, or updates it if a record with the same `customerId` already exists.

```typescript
const customer = await revstack.customers.identify(params);
```

<ParamField path="params" type="IdentifyCustomerParams" required>
  Customer identification parameters.
  
  <Expandable title="properties">
    <ParamField path="customerId" type="string" required>
      Unique customer identifier within your system.
      
      **Example:** `"user-123"`
    </ParamField>
    
    <ParamField path="email" type="string">
      Customer email address.
      
      **Example:** `"john@acme.com"`
    </ParamField>
    
    <ParamField path="name" type="string">
      Customer display name.
      
      **Example:** `"John Doe"`
    </ParamField>
    
    <ParamField path="metadata" type="Record<string, unknown>">
      Arbitrary metadata to attach to the customer.
      
      **Example:** `{ segment: "enterprise", referral: "partner" }`
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="customer" type="Customer">
  The created or updated customer record.
  
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique identifier with `usr_` prefix.
    </ResponseField>
    
    <ResponseField name="environmentId" type="string">
      Environment this customer belongs to.
    </ResponseField>
    
    <ResponseField name="email" type="string">
      Customer's email address.
    </ResponseField>
    
    <ResponseField name="providerUid" type="string">
      UID from external auth provider (e.g. Firebase, Auth0).
    </ResponseField>
    
    <ResponseField name="isAnonymous" type="boolean">
      Whether this customer was created anonymously.
    </ResponseField>
    
    <ResponseField name="gatewayCustomerId" type="string">
      External payment gateway customer ID (e.g. Stripe `cus_...`).
    </ResponseField>
    
    <ResponseField name="metadata" type="Record<string, unknown>">
      Arbitrary key-value metadata.
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      ISO 8601 timestamp of creation.
    </ResponseField>
  </Expandable>
</ResponseField>

**Example:**

```typescript
const customer = await revstack.customers.identify({
  customerId: "user-123",
  email: "john@acme.com",
  name: "John Doe",
  metadata: { segment: "enterprise" },
});
```

### get()

Retrieve a customer by ID.

```typescript
const customer = await revstack.customers.get(customerId);
```

<ParamField path="customerId" type="string" required>
  The customer's unique identifier.
  
  **Example:** `"usr_abc123"`
</ParamField>

<ResponseField name="customer" type="Customer">
  The customer record.
</ResponseField>

**Example:**

```typescript
const customer = await revstack.customers.get("usr_abc123");
console.log(customer.email); // "john@acme.com"
```

---

### list()

List customers with optional pagination.

```typescript
const response = await revstack.customers.list(params);
```

<ParamField path="params" type="ListParams">
  Pagination parameters.
  
  <Expandable title="properties">
    <ParamField path="limit" type="number">
      Maximum number of records to return.
      
      **Default:** `100`
    </ParamField>
    
    <ParamField path="offset" type="number">
      Number of records to skip.
      
      **Default:** `0`
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="response" type="PaginatedResponse<Customer>">
  A paginated list of customer records.
  
  <Expandable title="properties">
    <ResponseField name="data" type="Customer[]">
      Array of customer records.
    </ResponseField>
    
    <ResponseField name="total" type="number">
      Total number of records matching the query.
    </ResponseField>
    
    <ResponseField name="hasMore" type="boolean">
      Whether more records exist beyond this page.
    </ResponseField>
  </Expandable>
</ResponseField>

**Example:**

```typescript
const { data: customers, hasMore } = await revstack.customers.list({
  limit: 50,
  offset: 0,
});

console.log(`Found ${customers.length} customers`);
```

---

### update()

Update an existing customer's profile.

```typescript
const customer = await revstack.customers.update(customerId, params);
```

<ParamField path="customerId" type="string" required>
  The customer's unique identifier.
</ParamField>

<ParamField path="params" type="UpdateCustomerParams" required>
  Fields to update.
  
  <Expandable title="properties">
    <ParamField path="email" type="string">
      New email address.
    </ParamField>
    
    <ParamField path="metadata" type="Record<string, unknown>">
      Metadata fields to merge with existing metadata.
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="customer" type="Customer">
  The updated customer record.
</ResponseField>

**Example:**

```typescript
const customer = await revstack.customers.update("usr_abc123", {
  email: "newemail@acme.com",
  metadata: { verified: true },
});
```

---

### delete()

Delete a customer and all associated data.

```typescript
const result = await revstack.customers.delete(customerId);
```

<ParamField path="customerId" type="string" required>
  The customer's unique identifier.
</ParamField>

<ResponseField name="result" type="object">
  <ResponseField name="success" type="boolean">
    Confirmation of deletion.
  </ResponseField>
</ResponseField>

**Example:**

```typescript
const { success } = await revstack.customers.delete("usr_abc123");
if (success) {
  console.log("Customer deleted successfully");
}
```
